
分群列表
按照一定逻辑规则，抽取自定义范围内的用户，归纳为一组，并列表呈现。

功能分类
1.分群列表
2.删除分群
3.创建分群
点击创建分群按钮，进入分群页面

数据API
1.分群列表
进入列表页面，发送ajax，加载用户分群列表；分群数目不会很多，因此，
一次性加载全部分群

接收数据
{
	status:boolean//true-存在有效分群列表 false-空分群
	list:[{
		usrGroupName://分群名称
		usrCount：//分群人数
		//暂时定义两个常用分群
	}]//对象数组，每一个元素代表一个分群；无分群，返回空数组[]
}

2.删除分群
用户可删除后台的分群

发送数据
{
	usrGroupName://String,指定待删除的分组名
}
接受数据
{
	status://boolean,true-成功删除 false-删除失败
}
